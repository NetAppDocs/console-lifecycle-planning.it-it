== Accesso alla pianificazione del ciclo di vita

*Qual è l'URL della pianificazione del ciclo di vita?*

Per l'URL, in un browser, inserisci: https://console.netapp.com/["https://console.netapp.com/"^] per accedere alla NetApp Console.



== Licenza

*È necessaria una licenza per utilizzare la pianificazione del ciclo di vita?*

Non è richiesto un file di licenza NetApp (NLF).
